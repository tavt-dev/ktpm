# 

# \## 🔄 Quy trình kiểm thử áp dụng (STLC)

# 

# Dự án được thực hiện theo quy trình Software Testing Life Cycle:

# 

# 1\. \*\*Requirement Analysis\*\*

# &nbsp;  Phân tích tài liệu đặc tả, xác định phạm vi và chức năng cần kiểm thử.

# 

# 2\. \*\*Test Planning\*\*

# &nbsp;  Xây dựng Test Plan bao gồm:

# 

# &nbsp;  \* Scope

# &nbsp;  \* Objective

# &nbsp;  \* Strategy

# &nbsp;  \* Test Environment

# &nbsp;  \* Resource \& Schedule

# 

# 3\. \*\*Test Design\*\*

# &nbsp;  Thiết kế test case sử dụng các kỹ thuật:

# 

# &nbsp;  \* Equivalence Partitioning

# &nbsp;  \* Boundary Value Analysis

# &nbsp;  \* Negative Testing

# &nbsp;  \* Happy Path Testing

# 

# 4\. \*\*Test Execution\*\*

# &nbsp;  Thực thi test, ghi nhận:

# 

# &nbsp;  \* Actual Result

# &nbsp;  \* Pass/Fail

# &nbsp;  \* Evidence (Screenshot)

# 

# 5\. \*\*Defect Reporting\*\*

# &nbsp;  Ghi nhận lỗi theo chuẩn:

# 

# &nbsp;  \* Bug ID

# &nbsp;  \* Steps to Reproduce

# &nbsp;  \* Expected Result

# &nbsp;  \* Actual Result

# &nbsp;  \* Severity

# &nbsp;  \* Screenshot

# 

# 6\. \*\*Test Reporting\*\*

# &nbsp;  Tổng hợp:

# 

# &nbsp;  \* Tổng số test case

# &nbsp;  \* Tỷ lệ Pass/Fail

# &nbsp;  \* Phân loại lỗi theo mức độ

# &nbsp;  \* Đánh giá chất lượng hệ thống

# 

# ---

# 

# \## 👥 Cơ cấu nhóm (4 thành viên)

# 

# Nhóm được tổ chức theo mô hình phân công theo chức năng, đảm bảo mỗi thành viên chịu trách nhiệm toàn diện cho module của mình:

# 

# | Thành viên | Phụ trách                                   |

# | ---------- | ------------------------------------------- |

# | Member 1   | Đăng ký tài khoản                           |

# | Member 2   | Đăng nhập / Đăng xuất                       |

# | Member 3   | Giỏ hàng \& Đặt mua                          |

# | Member 4   | Bảo trì danh mục (Admin) \& Tổng hợp báo cáo |

# 

# Mỗi thành viên thực hiện đầy đủ:

# 

# \* Thiết kế test case

# \* Thực thi test

# \* Báo lỗi

# \* Cung cấp minh chứng

# 

# Toàn nhóm cùng review chéo và hoàn thiện báo cáo cuối cùng.
